# Artifact 2: Syscall Trace Analysis using k-window Modeling

## Overview
This artifact analysis Linux syscall traces under different execution scenarios using a **k-window model**. 
The objective is to study how syscall behaviour varies across **scenarios**,**execution modes**, and **window sizes (k)** using quantative metrics
